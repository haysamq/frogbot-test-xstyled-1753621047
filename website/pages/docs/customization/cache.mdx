# Cache

Customizing the cache behaviour of xstyled.

<carbon-ad />

## Turn off cache

By default xstyled cache every property resolution. If you need to disabled the cache behaviour, you can do it in the theme.

```js
export const theme = {
  xstyled: {
    cache: false,
  },
}
```
